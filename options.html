This plugin is designed to rapidly switch languages on Khan Academy without changing which content you're looking at, and without changing your account's language settings.

If the current content is not available in the selected language you'll get either a "not found" or "access denied" error.
You can select which languages you want to switch between by selecting them below. Then close this window and return to Khan Academy.

This extension was written by Adam Berkan at a Khan Academy Hackathon. The code is open source and hosted is hosted at https://github.com/khan/language-toggle. This software is provided as-is.
